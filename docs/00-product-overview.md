# 00 – Product Overview

Viral Clip AI mengubah 1 video panjang (YouTube / upload) menjadi puluhan short video berpotensi viral:
- Deteksi momen terbaik (hook, punchline, moment emosional).
- Skor virality per clip (0–10) dengan breakdown Hook / Flow / Value / Trend.
- Subtitle otomatis multi bahasa + style.
- Brand kit per user (warna, logo, watermark).
- Dub / voice-over AI + BGM.
- Export siap upload (Shorts / Reels / TikTok).

Tidak ada konsep project di UI: user cukup buka menu **AI Viral Clip**, upload / paste link, dan melihat daftar video di bawahnya.
